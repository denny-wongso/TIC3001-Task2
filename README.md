# TIC3001-Task2

# How to start

##Start the server side
1. Cd to server folder
1. run npm install
2. run npm start
3. go to localhost:8081/api/v1/dogs
4. you should be able to get the dogs

##Start the client sude
1. Go to client folder
2. Run npm install
3. Run npm start
4. Go to localhost:3000
